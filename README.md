# Sudoku Solver

## Usage

```sh
./SudokuSolver input_file.txt
```

## Input format

The input file can only contain numeric characters (`'0-9'`) and spaces:

```
5 0 1  0 6 0  2 0 9
7 3 0  0 2 0  0 5 4
9 0 0  8 0 0  0 0 7

0 0 0  0 0 0  3 0 0
2 5 0  0 0 0  0 6 8
0 0 7  0 0 0  0 0 0

6 0 0  0 0 9  0 0 3
4 1 0  0 5 0  0 2 6
8 0 9  0 1 0  4 0 5
```
